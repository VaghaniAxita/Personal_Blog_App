
# Personal Blog App 📝
A full-stack Personal Blog App that allows users to create, view, edit, and delete blog posts. Built with React.js for the frontend, Node.js with Express.js for the backend, and MongoDB for the database, this app is responsive, user-friendly.
## 📚 Features 

**1. Create Post:** Add new blog posts with a title and content.

**2. View All Posts:** Browse posts in reverse chronological order.

**3. View Post Details:** View full content of any blog post.

**4. Edit Post:** Update an existing post's title or content.

**5. Delete Post:** Remove unwanted posts permanently.

**6. Responsive Design:** Optimized for mobile and desktop devices.



## 🛠️ Technology Stack

**Backend:** 

- Node.js
- Express.js
- MongoDB


**Frontend:** 

- React.js 
- Axios
- React Router

**Database:**

- MongoDB: NoSQL database for data storage.
